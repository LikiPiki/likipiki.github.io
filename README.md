# Likipiki Blog

This is my [hugo](https://gohugo.io/) blog.

## Installation

1. Clone the repository by SHH or HTTP.
2. Install [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme:
	```
	cd likipiki.gitlab.io
	git submodule init
	git submodule update
	```
3. Run hugo server:
	```
	hugo server
	```
